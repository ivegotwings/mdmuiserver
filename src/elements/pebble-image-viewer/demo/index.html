A pebble-image-viewer

Use the sizing="cover" property to expand the image to cover all the specified dimensions.

Use the data:image/svg+xml,base64 as src attribute value.

Use the sizing="contain" property to expand the image to fit within the specified dimension.

Use the --pebble-image-width property to set the width of the image wrapped by the pebble-image-viewer.

Use the --pebble-image-height property to set the height of the image wrapped by the pebble-image-viewer.

No placeholder is shown by default.

The preload attribute shows a placeholder element in front of the image before it is loaded. Use the --iron-image-placeholder CSS mixin to style it.

Use the placeholder attribute to specify a background image for the placeholder element.

Use shape attribute to change the outline of the image thumbnail. Only valid value is circle.

Use big-image-src to upload a different image file to open on the click of the image thumbnail.
